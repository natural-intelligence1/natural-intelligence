# NI — Positioning Working Document

**Status:** Iterative thinking artefact, not a marketing document.
**Purpose:** Articulate what NI actually is, in plain language, before deciding how to describe it externally.
**Method:** Add to this document over time. Bring drafts back for critique. Resist the temptation to write hero copy until the underlying model is clear.

---

## How to use this document

This is not a manifesto, brand brief, or homepage draft. It is a thinking space.

Each section is a question. Answer it honestly, in your own words, without marketing language. If the answer is unclear or contested, write that down too. The point is to surface the underlying positioning model, not to commit to a polished version of it.

When a section feels genuinely answered (not just filled in), move on. When new thinking emerges that contradicts something already written, edit rather than appending — earlier drafts can live in git history.

The document is finished when reading it end-to-end produces a coherent picture of what NI is, why it exists, and who it's for. Hero copy and brand language come *after* that picture is clear.
